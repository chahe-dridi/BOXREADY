## ⚡️ Project Setup

### 1️⃣ **Clone the Repository**

```sh
git clone https://github.com/chahe-dridi/BOXREADY.git
cd BOXREADY
```

### 2️⃣ **Install Dependencies**

```sh
npm install
```

### 3️⃣ **Start Tailwind CSS Watch Mode**

```sh
npm run watch
```
